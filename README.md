This project is a RESTful Web API built using ASP.NET Core and Clean Architecture principles. The API provides authentication, user management, product management, and category management with standardized responses, validation, authorization, and global exception handling.

Technologies Used
ASP.NET Core Web API
Entity Framework Core
SQL Server
JWT Authentication
Clean Architecture
Swagger / OpenAPI
Dependency Injection
